Open Source Software

Here is a list of software applications that I have found to be adequate if not excellent replacements for their non-free counterparts:

  • Debian GNU/Linux - Debian is a free operating system (OS) for your computer. It is one of the best (in my not so humble opinion) operating systems available.
  • Dia - A diagramming and modeling tool. It can be used as a replacement for the Microsoft Visio drawing program though it is more limited (a good thing in my opinion as Visio is bloated).
  • FileZilla - A FTP program. This is a very solid FTP program that runs on both GNU/Linux and Windows.
  • Firefox - A web browser Mozilla put out by the Mozilla foundation.
  • Open Office - A direct replacement for Microsoft Office. It has Writer (replacement for Word), Calc (relacement for Excel), Impress (replacement for Powerpoint), Draw (diagrams, 3D illustrations).
  • Seamonkey - A replacement for Microsoft Internet Explorer. It has many features that Internet Explorer has been lacking for quite some time. Includes: Web browser, email client, news client, address book, and bookmark manager.
  • The Gimp - An excellent image editing package, similar to Adobe Photoshop replacement.
  • Thunderbird - An email/news/rss client put out by the Mozilla foundation.
  • 7-Zip - A file archiver with excellent compression ratio, supports many formats, and runs on GNU/Linux and Windows.
