I am an advocate for Free/Open Source Software. It is not about the software being free as in beer (which means you don't pay for it), it is about free as in speech (which means it is a right that you have that cannot be taken from you). I use the Debian GNU/Linux operating system. Not only is it an amazing collection of software based on GNU/Linux, it also upholds the tenets of the FOSS community. It is in fact, the most "Free" operating system available.
